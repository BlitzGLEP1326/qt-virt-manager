qt-virt-manager

is the GUI for (remote) control a Virtual machines and entities. The application is designed for (semi-)professional usage.

Uses Qt as framework for GUI.

Uses libvirt as the back-end management API for control a virtual entities.

Uses Spice client viewer for graphic consoles to guest domains (VMs).

Uses VNC client viewer for graphic consoles to guest domains (VMs).

Development occurs based on the package database of Fedora.

Application is present as package

  • in repo: dnf install qt-virt-manager (Qt5 only)

  • in Fedora Copr: dnf install qt4-virt-manager or dnf install qt5-virt-manager

  • in FedoraPeople

The Main window for control state and common operations with connections and related (aka virtual entities) virtual machines (VM), networks, storages, interfaces, secrets.

The VM Settings window for create/edit a virtual machines (domains).

The SPICE/VNC VM Viewer for graphical control a VMs with support graphic channel.

The LXC VM Viewer for graphical control a lxc-containers.

The Task Warehouse window for viewing states of current tasks.

Does not contain platform-specific components.
